# Chat-Application
Multithread Chat App with a server and clients

The app represents a messenger chat. It was done individually as a small assignment. The app allows to create a room with a server and clients. The messages can be sent between a server and clients, or between clients. The application is multithreaded, meaning all parts of the app (server and every client) work simultaneously. Languages and skills utilized: Java, Java Threads, IntelliJ.
